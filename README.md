# Character Explorer

A lightweight desktop application built with Python and PyQt6 to manage and archive your characters. It features a local SQLite database, a searchable interface, and a detailed character view.

## Features

- **Character Management**: Add, edit, and delete characters with ease.
- **Rich Details**: Store names, age, creator info, and detailed descriptions.
- **Image Support**: Upload and store character images directly in the local database.
- **OC Tracking**: Mark characters as "Original Characters" (OC).
- **Franchise System**: Organize characters by their respective franchises.
- **Live Search**: Quickly filter through your collection using the integrated real-time search bar.
- **Local Storage**: All data is saved locally on your machine in a SQLite database.
